How to Run:
Git clone the file
(this project runs of the combined power of python server and 
a node server, for the use of socket.io)

To run the file localy you need to run 
python manage.py runserver
in the main file location (Where manage.py sits)

$ pip install -r requirements.txt --no-index --find-links file:///tmp/packages

To run the node server, first make sure you have node installed
second traverse into the nodejs file where the file index.js can be found
than on your termanal type 
node index.js
To start the node server.

That should run your server on localhost:8000




P.S.
The node modules are packaged in the github file
